# 🎵 Emotion-Based Music Player

## 📌 Project Description

This project detects the user's emotion using a webcam and recommends songs based on the detected emotion.

## 🚀 Features

* Real-time face detection using OpenCV
* Emotion prediction using Deep Learning model
* Music recommendation based on emotion
* Supports multiple emotions (Happy, Sad, Angry, Neutral)

## 🛠️ Tech Stack

* Python
* OpenCV
* TensorFlow / Keras
* Pandas

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the project:
   python capture.py
